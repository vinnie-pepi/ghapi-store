A quick intro to creating your template:

  1. Add some sample json data to the sample data folder here: [ link to sample data ]
  2. Sample data can be accessed within your template by using the handlebars templating system:
    http://handlebarsjs.com/

    You can think of the sample data as your context in handlebars so for example, the sample form below uses the following sample data: 

            {
          "name": "Factual Shanghai Office",
          "address": "166 Kaixuan Road Suite 101, Building 1",
          "open": "YES",
          "type": "office",
          "number": "123456",
          "notes": "hello world!"
        }
  3. Build your form using html and data from your sample to however you like it. (Bootstrap and Jquery included)
  4. Gather any inputs you like from the worker. What you name your inputs will be how they are stored in mastermind and how you will access them later. If a worker comes back to this template, the inputs that he or she had previously saved will be populated so there is no need to set a value on a text input or set checked on a radio or a checkbox. The submit buttons will also be created for you.
  5. Finally, you can access these instructions anytime here: [ add link here ] so feel free to delete this page and start from a clean slate.
